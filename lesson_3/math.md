# 數學筆記（國中程度）

## 目標
- 解一元一次方程式
- 進行分數加減
- 了解比與比例

## 重點
- 一元一次方程式：`ax + b = c`
- 分數通分：找最小公倍數
- 比例：`a : b = c : d` 表示 `a/b = c/d`

## 範例
1. 解方程式：`3x + 5 = 20`  
   `3x = 15` -> `x = 5`
2. 分數計算：`1/4 + 1/6`  
   `3/12 + 2/12 = 5/12`

## 練習
1. `2x - 7 = 9`，求 `x`。
2. `3/8 + 1/4`，化為最簡分數。
3. 若 `a : b = 2 : 3`，且 `a = 8`，求 `b`。
4. 解方程式：`5x + 2 = 3x + 10`
